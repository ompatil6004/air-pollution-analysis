# Air Pollution Analysis

This project analyzes air pollution data using Python and visualizes trends through dashboards to understand environmental patterns across cities.

## Tools Used
- Python (Pandas, NumPy)
- Data Visualization
- Power BI Dashboard

## Key Insights
- Compared pollution levels across locations
- Identified high-pollution regions and trends
- Created dashboards for visual environmental monitoring

## Objective
To demonstrate practical data analysis and visualization skills for entry-level Data Analyst roles.
